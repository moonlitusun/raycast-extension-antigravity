## Setup

1. Install the extension in Raycast
2. Set your projects directory in preferences (default: `/Users/rocsun/Code`)
3. Adjust search depth if needed (default: 3 levels)
4. Make sure `agy` command is available in your PATH

## Usage

1. Open Raycast
2. Type "Search Antigravity Projects"
3. Start typing to filter projects
4. Press Enter to open the selected project with Antigravity

## Actions

- **Enter**: Open project with Antigravity
- **Cmd+Enter**: Show in Finder
- **Cmd+C**: Copy project path to clipboard

## Requirements

- Antigravity IDE installed
- `agy` command available in PATH
